SaddleBrook Senior Softball (SBSS) Data Viewer

Version 1.15.24244 — Released Date Saturday, August 31 2024
The SBSS Data Viewer lets you view accumulated statistics from games played. All this data is gathered from the public and official SaddleBrooke Senior Softball Association (SSSA) site.
The data from the current season are updated every weekday evening no later than 6:30 PM, after which you can see the all pages populated using the latest data. Data from other seasons can be viewed by selecting the season from the drop-down list when clicking the "Seasons" button.

About SBSS Data Viewer Tables The page is a detailed look at the features of Data Viewer tables using live tables that you view and test while reading the page. A must read if you haven't yet.
Game Results, Teams and Players This page contains all the data gathered from the SSSA site, which is stored in a file (called the Data Store) and is displayed using three perspectives.
  • For every played game not canceled or forfeited, the results for the teams and each of the players on the team are displayed. This is a lot of data!
  • Second, there is a table that views the results from the team perspective, so it shows the team standings and the player summaries for every game the team played.
  • Finally, the last perspective is from the player. All stats (including ranking) for every player in every team in the league is displayed. The data for all leagues (typically 8 or 9) are displayed.
Player Sheets Player Sheets focus directly on each player. Every player has their own "sheet" that has their summary stats for every league in which they played as well as rank, percentile and Z-score statistics.

Viewing Requirements

This Web site is best viewed in a desktop browser because of the size of the tables that can be nested and the number of tables. All pages render on a phone but can be hard to navigate and see everything. This has only been tested on chromium browsers (Chrome and Edge) and Android and iPhones. Your mileage may vary.
When you select a link to a page, if it is displayed same tab, to go back previous page just click on the link that is part of the page header. See the Things to Note & Information page for more detailed information about the site structure.

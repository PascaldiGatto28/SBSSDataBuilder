SaddleBrook Senior Softball (SBSS) Data Viewer

Version 1.12.24204 — Released Date Monday, July 22 2024
The SBSS Data Viewer lets you view accumulated statistics from games played. All this data is gathered from the public and official SaddleBrooke Senior Softball Association (SSSA) site.
The data from the current season are updated every weekday evening no later than 6:30 PM, after which you can see the all pages populated using the latest data.

Game Results, Teams and Players
This page contains all the data gathered from the SSSA site, which is stored in a file (called the Data Store) and is displayed using three perspectives.
  • For every played game not canceled or forfeited, the results for the teams and each of the players on the team are displayed. This is a lot of data!
  • Second, there is a table that views the results from the team perspective, so it shows the team standings and the player summaries for every game the team played.
  • Finally, the last perspective is from the player. All stats (including ranking) for every player in every team in the league is displayed. The data for all leagues (typically 8 or 9) are displayed.

Using the Data Viewer Tables

Data is presented as tables, and when there's a lot of data it can be hard to navigate. But all tables can be collapsed or expanded by clicking on the table header anywhere within the text of the table header. And when there are a lot of tables, buttons are available to collapse or expand all tables on the page. Try clicking on the table header above. When a table is expanded, the arrow to the left of the header text is up and the bottom border of the table is solid. If the arrow is down and the bottom border is dashed, the table is collapsed and can be expanded, by clicking on the header text. Finally the table column header text is "sticky" — when you scroll down, and the table is visible, the table header remain attached to the top of browser until you scroll past the table.

Tool Tips

The tables are composed of a title and column headers and columns of data. The column header text (for example player stats, like AB, TH, OPS, etc.) are documented in the ""Definitions, Explanations & Details"" page. For all tables when you hover over the column header text, a tool tip pops open for a few seconds. This can be quite helpful if you're not sure what the data represents. Be aware that tool tips are only displayed for table column header text in general. There are exceptions where table values actually have tool tips, and that's for Percentiles, Rankings, Z-scores and league summary in the "Player Sheets" page. This is done because the information is inserted in the player's table only when the data is relevant (that is, enough plate appearances to be part of the calculations) and is not really part of the table.

Sortable Tables

Some player tables are sortable. When a page is displayed you see a down-arrow icon indicating that the table is ordered in descending ordered based upon the values in that column. If you click on the column again, that table is displayed in ascending order. Here's what see if PA (Plate Appearances) is the initial column when page is displayed and then when it is clicked again:
If you click another (numeric) column, the icons on the current column disappear and you see the down arrow and the table is sorted in descending order by the values in the column. The player name column is just the opposite: When you first click on it the table is ordered in ascending order by player last (then first name). If you click on it again, the table is order in descending order by player last name (then first name).
As of this writing, the only sortable player stats table in on the Players Leagues Stats & Statistics page, but that may be extended as needed in future releases.

Viewing Requirements

This Web site is best viewed in a desktop browser because of the size of the tables that can be nested and the number of tables. All pages render on a phone but can be hard to navigate and see everything. This has only been tested on chromium browsers (Chrome and Edge) and Android and iPhones. Your mileage may vary.
When you select a link to a page, if it is displayed same tab, to go back previous page just click on the link that is part of the page header. See the Notes & Information page for more detailed information about the site structure.

Games Results, Team and Players Stats

SaddleBrooke Senior Softball data is generated in the following ways:
  • Before any games are even played, the entire schedule is published as a list of scheduled games for each of the leagues for the season.
  • As games are played (or canceled), the results of the game together with player box scores for each team are posted.
  • For each league, team standings (wins, losses, runs scored and runs against) are updated as well.
  • For each team, cumulative player stats are recorded in the Rosters & Stats pages including computed stats like batting and slugging averages, on base percentage and on-base percentage plus slugging.
All the updates to the data originate from the volunteer score keepers and then posted to the SaddleBrooke Senior Softball Association (SSSA) web site by the site manager. It's an incredible amount work that's very well done.
The SBSS data builder application "scrapes" the pages of the SSSA site to recover the day's data and stores it in a file (a data store, we call it). The advantage is that all the data is in a single file that can be searched and results displayed. Moreover, additional computed computer from what's already in the data store can be computed and displayed.
There are really three fundamental ways of looking at this data: (1) From the perspective of game results; (2) Focus on the teams; (3) From the players' view. The purpose of this page is put all this information in a three main tables for each of leagues aligning with the different perspectives. In addition, you'll see a table that just describes the current state of the data store (file). As of the this writing there are over 1,100 tables representing the current season, some of which are nested deeply in other tables. That volume is driver for allowing tables collapsed to make navigation easier.
Before diving and looking at all the data, you might want to take a look at details as we guide you through the different tables. When describing a table, you can open to follow, or just click on the Show me ... link, and it open automatically for you.

For each league, the first table contains the game results, for example "Game Results Data for the 16 Monday Community Games". For each scheduled game (row of the table), in the first you can see schedule detail In the second column is another table having the games results, and for each team, the nested tables are score cards for each players.
Although the extended statistics are not terribly meaningful for a single game, they are included for consistency other tables represent summary data. There is a field is not displayed in the SSSA site, and that plate appearances (PA). This is used by the player ranking stats (see below).
Also note that data for player participating are displayed even if the player is no longer part of the softball program. In particular that means the data for the teams always reflect the individual player data.

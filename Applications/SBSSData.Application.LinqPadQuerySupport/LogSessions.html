Data Store Log (Back to SBSS Data Viewer)

The Data Store Information table describes the state of the file where all information is stored (the Data Store). It is updated every weekday evening at 8:30 PM. This table gives you an idea of the volume of data and how it's changing.
Following is a log of all the changes that are made to the Data Store during those nightly updates. It's used to make sure that operations are working correctly, but it is interesting it its own right. The system is robust and provides confidence that all the data is recovered correctly.

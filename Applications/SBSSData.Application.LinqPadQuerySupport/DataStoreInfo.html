Data Store Information (Back to SBSS Data Viewer)

The Data Store Information table describes the state of the file (Data Store) where all information gathered from the SSSA is stored.
The following table contains all the information for the most recent seasons, the current season being the first row. The information for the current season is updated every weekday no later than 6:30 PM. This table gives you an idea of the season activity and how it's changing.
The remaining rows are of the most recent seasons completed. They provide some historical data of the activity in the past as compared to the current season.

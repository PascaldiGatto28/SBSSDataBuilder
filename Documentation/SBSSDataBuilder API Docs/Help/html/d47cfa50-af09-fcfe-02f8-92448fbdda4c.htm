      Parameters

      leagueScheduleLocation  String
      The location of a league is the string representation of a Uri to a page having links to all the scheduled games. For example https://saddlebrookesoftball.com/calendar/monday-recreation-schedule/ is the link to the schedule for the Monday Recreation league.

      Return Value

      LeagueSchedule
      A LeagueSchedule instance. If leagueScheduleLocation is null or empty, the empty LeagueSchedule instance (that is, IsEmpty is true) is returned. null is never returned.

      Remarks

      Short of using reflection, this is the only way to create an instance of a LeagueSchedule class. From the page recovered using the leagueScheduleLocation property, the contents are scraped to initialize the LeagueDescription property, and follow on the schedule game links on the page to build the ScheduledGames property.

      LeagueDescription Class

      Encapsulates meta-information about a league which is encapsulated in the LeagueSchedule class when it's constructed.

      Remarks

      When a LeagueSchedule is created by the static method ConstructLeagueSchedule(String) a description of the league including the league schedule page is also constructed. This class is often used for reporting purposes. Instances are only constructed via the static method or JSON deserialization.

      Because the default constructor is private and all property setters are "private init", instance of this class are essentially immutable and typically can only be created using the static ConstructionLeagueDescription(Uri, HtmlDocument) method.

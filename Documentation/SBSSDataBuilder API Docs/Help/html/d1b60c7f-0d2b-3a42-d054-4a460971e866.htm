      Parameters

      content  String
      The string from which the substring is returned. If null or empty, the empty string is returned.
      start  String
      The returned substring starts at the location of the value of this parameter.
      end  String
      The returned substring ends at the location of the value of this parameter.
      includeStart  Boolean  (Optional)
      If true, the start string is included in the returned substring; otherwise the returned substring begins immediated after the start string. The default is false.
      includeEnd  Boolean  (Optional)
      If true, the end string is included in the returned substring; otherwise the returned substring ends at the character immediately before the end string. The default is true.

      Return Value

      String
      A substring based upon the values of the parameters. If the source (content), start, end are null or empty, the empty string is returned. The empty string is also returned if an ArgumentOutOfRangeException object is thrown; this can happen if the start string does not appear before the end string. null is never returned.

      Usage Note

      In Visual Basic and C#, you can call this method as an instance method on any object of type String. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
